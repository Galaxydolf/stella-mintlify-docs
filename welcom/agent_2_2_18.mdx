## データ可視化

![](https://uxkpl4cba3j.feishu.cn/space/api/box/stream/download/asynccode/?code=MTEwYTFlNWY4ODFkZGY3MDQ2MDJlMTk5YzM0NWI4ZDNfUFc4MTRkS2ZZVkxNeE1JTkVVWFUzQjNVWkhZS3BLSzBfVG9rZW46UWh4VGJTOElQb1lkQTd4U216U2NuclVNblllXzE3NDMwNDMxNjI6MTc0MzA0Njc2Ml9WNA)

**モジュール概要**

* 「データ可視化」モジュールは、**データベースクエリ結果をグラフとして可視化する**ためのコンポーネントです。通常、「データベースクエリ」モジュールと組み合わせて使用します。指定された形式のチャート（棒グラフ、横棒グラフなど）を自動生成し、視覚的にデータを理解しやすくします。



**モジュールパラメータ**

| パラメータ名            | タイプ      | 機能説明                                                         |
| ----------------- | -------- | ------------------------------------------------------------ |
| **連動アクティベーション**   | ブール型     | 複数のブール型入力と接続可能。**すべての条件がTrueの場合に実行**されます（AND条件）。             |
| **任意条件アクティベーション** | ブール型     | 複数のブール型入力と接続可能。**いずれかの条件がTrueの場合に実行**されます（OR条件）。             |
| **元データ**          | テキスト型    | 「データベースクエリ」モジュールから出力されたクエリ結果を文字列として受け取るフィールド。                |
| **グラフタイプ**        | -        | 棒グラフ・横棒グラフなどから必要なチャートタイプを選択。&#xA;表示後、**ユーザー対話画面で再編集・変更も可能**。 |
| **モジュール実行完了**     | **ブール値** | モジュール処理完了後に **True** を出力します。通常は次の処理のトリガーとして使用します。            |
