## コードブロック

<img
  src="/images/29.png"
/>
**モジュール概要**

* JavaScript または Python のコードを追加し、データ処理を実行できる

**モジュールパラメータ**

| パラメータ名            | データ形式 | 機能説明                                                                                                                                            |
| :------------------------------ | :------: | :-------------------------------------------------- |
| **連動アクティベーション**   | ブール型  | 本ノードは複数のブール型入力を接続可能。**全ての条件が `true`&#x20;**&#x20;の場合に本モジュールを実行（例：　「すべての条件が満たされた場合に実行」 ）                                                         |
| **任意条件アクティベーション** | ブール型  | 本ノードは複数のブール型入力を接続可能。**いずれか1つでも条件が `true`&#x20;**&#x306E;場合に本モジュールを実行（例：　「いずれか1つでも条件を満した場合に実行」 ）                                                 |
| **入力パラメータの追加**    | 複数型   | 処理対象の入力パラメータを設定（ブール型、文字列型、ナレッジベース検索結果型をサポート）                                                                                                    |
| **言語選択**          | -     | 使用するプログラミング言語を選択（JavaScript または Python）                                                                                                         |
| **コードの説明**        | -     | 処理内容をメモとして記述（オプション）                                                                                                                             |
| **コード内容**         | -     | ユーザーが記述するコード本体を入力・**Python の場合、関数名は `userFunction` に指定する必要有り**・**入出力は Key-Value データ形式で、Key は文字列型（String）**・**Key は設定した入力パラメータ・出力パラメータのキーに対応する** |
| **出力パラメータの追加**    | 複数型   | 処理結果の出力パラメータを設定（ブール型、文字列型、ナレッジベース検索結果型をサポート）                                                                                                    |
| **実行成功**          | ブール型  | コードが正常に実行された場合、`True` を出力                                                                                                                       |
| **実行エラー**         | ブール型  | コード実行時にエラーが発生した場合、`True` を出力                                                                                                                    |
| **実行結果**          | 文字列型  | コード実行の全結果を出力。複数の出力パラメータがある場合、JSON 形式で統合して出力                                                                                                     |
| **モジュール実行完了**     | ブール型  | **本モジュールの実行完了時、出力値は `True` となる**（通常、次処理を開始するためのトリガーとして使用）                                                                                       |
