## データベースクエリ

<img
  src="/images/35.png"
/>
**モジュール概要**

* 「データベースクエリ」モジュールは、**ワークフロー内でデータベースに対するSQLクエリの実行**を簡単に実現するためのコンポーネントです。あらかじめ接続済みのデータベースを指定し、SQLを実行し、結果を取得することができます。

**モジュールパラメータ**

| パラメータ名            | タイプ   | 機能説明                                                                             |
| :------------------------------ | :------: | :-------------------------------------------------- |
| **連動アクティベーション**   | ブール型  | 複数のブール型入力と接続可能。**すべての条件がTrueの場合に実行**されます（AND条件）。                                 |
| **任意条件アクティベーション** | ブール型  | 複数のブール型入力と接続可能。**いずれかの条件がTrueの場合に実行**されます（OR条件）。                                 |
| **SQL**           | テキスト型 | 実行したいSQL文を入力。内容は、モデルが自動生成したSQLでも、あらかじめ固定されたSQLでも構いません。要件に応じて柔軟に設定できます。           |
| クエリ対象データベース       | -     | このクエリを実行する対象のデータベースを選択。&#xA;使用前に、**インテリジェントエージェントの設定画面で該当DBとの接続を確立**しておく必要があります。 |
| **クエリ結果**         | テキスト型 | SQLの実行結果。次の処理や表示に活用するため、文字列型で出力されます。                                             |
| **実行成功**          | ブール型  | SQL実行に成功した場合、**True** を出力。通常、後続処理のトリガーとして使用します。                                  |
| **実行失敗**          | ブール型  | SQL実行に失敗した場合、**True** を出力。エラーハンドリングに使用します。                                       |
| **モジュール完了通知**     | ブール型  | 処理完了後、**True** を出力。後続の処理を続けるためのトリガーとして使用。                                        |
| **クエリ結果を表示するか**   | -     | エンドユーザーとの対話画面で、クエリ結果を表示するかどうかの設定（チェックON/OFF）。                                    |
