## **情報分類（情報のカテゴリー分け）**

<img
  src="/images/19.png"
/>
**モジュール概要**

AIのインテリジェント分析を活用し、ユーザーの質問（入力）を分類します。これにより、異なる種類の質問に対して適切な処理を実行し、よりパーソナライズされた対応が可能になります。

**モジュールパラメータ概要**

| パラメータ名            | データ形式        | 機能説明                                                                                            |
| ----------------- | ------------ | ----------------------------------------------------------------------------------------------- |
| **連動アクティベーション**   | ブール型         | 本ノードは複数のブール型入力を接続可能。**全ての条件が `true`&#x20;**&#x20;の場合に本モジュールを実行（例：　「すべての条件が満たされた場合に実行」 ）         |
| **任意条件アクティベーション** | ブール型         | 本ノードは複数のブール型入力を接続可能。**いずれか1つでも条件が `true`&#x20;**&#x306E;場合に本モジュールを実行（例：　「いずれか1つでも条件を満した場合に実行」 ） |
| **情報入力**          | 文字列型         | 前の処理で生成されたテキストをAIモデルに入力                                                                         |
| **ナレッジベース検索結果**   | ナレッジベース検索結果型 | ナレッジベース検索モジュールの検索結果を入力として使用                                                                     |
| **チャットコンテキスト**    | -            | ・過去のチャット履歴をAIモデルに渡し、文脈を考慮した解析を実行・**0～6件のチャット履歴**を情報入力として設定可能                                    |
| **モデル選択**         | -            | ・本モジュールで使用する AI**モデル** を選択可能・異なるモジュールごとに、異なるAIモデルを設定可能                                          |
| **システムプロンプト**     | -            | ・大規模言語モデルに対して、処理すべきタスクを**自然言語（Prompt）**&#x3067;指示                                               |
| **タグ追加**          | -            | ・ユーザーの入力情報をAIが分類し、該当する **タグ** に出力・1つの入力を **複数のタグ** に分類でき、後続モジュールへ接続して処理**例:**                   |
| **タグノード（出力）**     | ブール型         | ユーザー入力が該当するカテゴリーに分類された場合、該当タグの値は `True` となる／該当しない場合は `False` となる                                |
| **分類結果**          | 文字列型         | **JSON形式** で分類結果を出力                                                                             |
| **モジュール実行完了**     | ブール型         | **本モジュールの実行完了時、出力値は `True` となる**（通常、次処理を開始するためのトリガーとして使用）                                       |
