**モジュール概要**

* 「コード抽出器」モジュールは、**大規模言語モデル（LLM）から生成された出力の中から、特定のコード部分のみを抽出**するためのコンポーネントです。
  &#x20;対象は **Markdown 形式で記述されたコードブロック** に限定されており、自然言語と混在した出力からも、指定した言語のコードのみを正確に取得できます。



**モジュールパラメータ**

| パラメータ名            | タイプ   | 機能説明                                                                                  |
| ----------------- | ----- | ------------------------------------------------------------------------------------- |
| **連動アクティベーション**   | ブール型  | 複数のブール型入力と接続可能。**すべての条件がTrueの場合に実行**されます（AND条件）。                                      |
| **任意条件アクティベーション** | ブール型  | 複数のブール型入力と接続可能。**いずれかの条件がTrueの場合に実行**されます（OR条件）。                                      |
| **Markdown**      | テキスト型 | 上流のLLM出力から受け取る **Markdown形式の全文**。コードブロックの抽出対象。                                        |
| **コードタイプ**        | -     | 抽出したいプログラミング言語を指定。以下の言語をサポート：&#xA;**SQL / Python / Java / C / JSON / JavaScript（JS）** |
| **モジュール実行完了**     | ブール型  | モジュール処理完了後に **True** を出力します。通常は次の処理のトリガーとして使用します。                                     |
