# エージェント概要

このエージェントは、**社内文書（社員ハンドブックなど）を対象に開発された社内情報検索アシスタント**です。
入社前後の社員が、会社の規則や業務フローについてすばやく把握できるよう、**AIによる質疑応答形式**でわかりやすく情報を提供し、制度理解の促進と業務適応のスピードアップをサポートします。

# 開発背景

社員が会社のルールや手続きについて調べる際、情報の断片化や検索の手間が課題となっていました。
このエージェントは、**問い合わせの時間コストを削減し、誰でも迷わず社内情報にアクセスできる環境**を提供するために開発されました。個別ニーズに応じた回答を即時に返すことで、全社員のスムーズな業務開始を支援します。

# 使用方法

質問をチャット欄に入力するだけで、即座に回答を得られます。例：

* 勤務ルールについて知りたい場合： 「会社の出勤・退勤のルールはどうなっていますか？」

* 休暇申請について聞きたい場合： 「年次有給休暇はどのように申請しますか？」

* 出張手続きについて知りたい場合： 「出張の際に事前に提出すべき書類はありますか？」

このような質問に、エージェントがリアルタイムで明確に回答します。
