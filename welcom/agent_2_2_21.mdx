## **確認返信**

<img
  src="/images/39.png"
/>
**モジュール概要**

「確認返信」モジュールは、特定のテキストメッセージをユーザーに返答するためのコンポーネントです。あらかじめ設定されたメッセージを表示させるほか、外部ノードから入力を受けた場合、その入力内容が優先されてユーザーに表示されます。



**モジュールパラメータ**

| パラメータ名            | タイプ  | 機能説明                                                          |
| :------------------------------ | :------: | :-------------------------------------------------- |
| **連動アクティベーション**   | ブール型 | 複数のブール型入力と接続可能。**すべての条件がTrueの場合に実行**されます（AND条件）。              |
| **任意条件アクティベーション** | ブール型 | 複数のブール型入力と接続可能。**いずれかの条件がTrueの場合に実行**されます（OR条件）。              |
| **応答表示（ユーザー向け）**  | -    | このモジュールの出力をユーザーに表示するかどうかを制御。中間処理では非表示も可。                      |
| **応答内容（入力）**      | 文字列型 | 他ノードから入力がある場合、その内容をそのままユーザーに出力。入力がない場合、ここに記載したテキストがユーザーに返される。 |
| **応答内容（出力）**      | 文字列型 | 上記いずれかの内容が出力され、他モジュールへの接続も可能。                                 |
| **モジュール実行完了**     | ブール型 | モジュール処理完了後に **True** を出力します。通常は次の処理のトリガーとして使用します。             |
