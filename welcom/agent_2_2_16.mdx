## ドキュメント出力

<img
  src="/images/83.png"
/>

モジュール**概要**

「ドキュメント出力」モジュールは、**上流モジュールで生成された内容を Word 形式のドキュメント（.docx）として出力**するためのコンポーネントです。

また、**プレースホルダー（変数）を含む Word テンプレート**をアップロードすることで、指定した値を文中に自動で差し込むことも可能です。



**モジュールパラメータ**

| パラメータ名            | タイプ       | 機能説明                                                                               |
| ----------------- | --------- | ---------------------------------------------------------------------------------- |
| **連動アクティベーション**   | **ブール値**  | 複数のブール型入力と接続可能。すべての条件が **True** の場合にこのモジュールを実行します。つまり「**すべての条件が満たされたときに実行**」されます。  |
| **任意条件アクティベーション** | **ブール値**  | 複数のブール型入力と接続可能。**いずれかの条件が True** の場合に実行します。つまり「**どれか1つでも条件を満たせば実行**」されます。          |
| **情報入力**          | **テキスト型** | Wordテンプレートが未設定の場合、ここに入力された情報が **空白のドキュメントにそのまま挿入** されます。                           |
| **カスタム情報入力追加**    | **任意型**   | Wordテンプレートが設定されている場合に使用。プレースホルダー用のキーと名称を設定します。テンプレート内に複数のプレースホルダーがある場合、すべて登録が必要です。 |
| **モジュール実行完了**     | **ブール値**  | モジュール処理完了後に **True** を出力します。通常は次の処理のトリガーとして使用します。                                  |
