![](https://uxkpl4cba3j.feishu.cn/space/api/box/stream/download/asynccode/?code=YWRlYTA0OTUyYWEzZmViZWZmNGM1ODU5MmMxZGVmNWZfMGZpRG5ZUG1xM0JEbEJ2clFpd2JpckpCOXFHY3hHaUZfVG9rZW46V3M2OWJycTg4bzNpVEt4bDZnTmM4ZXdhblRkXzE3NDI5OTcwOTE6MTc0MzAwMDY5MV9WNA)

**モジュール概要**

ドキュメントの読み取りと審査を行うモジュール。ユーザーがドキュメントをアップロードすると、モデルが内容を解析し、あらかじめ設定された質問に従い順番に回答する

**モジュールパラメータ**

| パラメータ名            | データ形式        | 機能説明                                                                                                             |
| ----------------- | ------------ | ---------------------------------------------------------------------------------------------------------------- |
| **連動アクティベーション**   | ブール型         | 本ノードは複数のブール型入力を接続可能。**全ての条件が `true`&#x20;**&#x20;の場合に本モジュールを実行（例：　「すべての条件が満たされた場合に実行」 ）                          |
| **任意条件アクティベーション** | ブール型         | 本ノードは複数のブール型入力を接続可能。**いずれか1つでも条件が `true`&#x20;**&#x306E;場合に本モジュールを実行（例：　「いずれか1つでも条件を満した場合に実行」 ）                  |
| **ドキュメント情報**      | 文字列型         | ・ 「ユーザー入力」モジュールの「ドキュメント情報」から出力されたデータを受け取る・使用には「ユーザー入力」モジュールの「ドキュメントレビュー」を有効にし、入力フィールド左側にドキュメントアップロードボタンを追加してください |
| **一括質問**          | -            | ・審査する入力を設定する（例：「この契約書の甲は誰か？」など）・システムはプロンプト＋質問＋ドキュメント情報を入力として処理し、回答を生成（最大100件の質問を一括で設定可能）                         |
| **テンプレート追加**      | -            | CSVファイルを使用して、複数の質問を一括インポート可能                                                                                     |
| **モデル選択**         | -            | ・本モジュールで使用する AI**モデル** を選択可能・異なるモジュールごとに、異なるAIモデルを設定可能                                                           |
| **システムプロンプト**     | -            | AIモデルに対して、処理すべきタスクを**自然言語（Prompt）**&#x3067;指示                                                                    |
| **応答終了**          | ブール型         | すべての処理が完了すると `はい / いいえ` を出力し、次のコンポーネントの実行条件として利用可能                                                               |
| **応答内容**          | ナレッジベース検索結果型 | AIが生成した回答を出力。次のモジュールの入力データとして使用することも可能                                                                           |
| **モジュール実行完了**     | ブール型         | **本モジュールの実行完了時、出力値は `True` となる**（通常、次処理を開始するためのトリガーとして使用）                                                        |
