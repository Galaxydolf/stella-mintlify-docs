**モジュール概要**

「インテリジェントチャット」モジュールは、大規模言語モデル（LLM）の能力を活用し、ユーザーからの入力に対して自然言語による応答を生成するコンポーネントです。入力情報・プロンプト・コンテキストをもとに、柔軟かつ正確な対話体験を構築できます。



**モジュールパラメータ**

| パラメータ名            | タイプ            | 機能説明                                              |
| ----------------- | -------------- | ------------------------------------------------- |
| **連動アクティベーション**   | ブール型           | 複数のブール型入力と接続可能。**すべての条件がTrueの場合に実行**されます（AND条件）。  |
| **任意条件アクティベーション** | ブール型           | 複数のブール型入力と接続可能。**いずれかの条件がTrueの場合に実行**されます（OR条件）。  |
| **情報入力**          | 文字列型           | ユーザーの入力テキスト。上流モジュールの出力を接続して、LLMに渡す。               |
| **ナレッジベース検索結果**   | ナレッジベース検索結果タイプ | ナレッジベース検索モジュールの出力を接続。追加情報としてモデルに渡す。               |
| **チャットコンテキスト**    | -              | 直近の会話履歴（0〜6件）をモデルに入力。文脈理解を深め、より適切な応答を生成。          |
| **使用モデル**         | -              | 使用する大規模言語モデルを選択。モジュール単位で異なるモデルを指定可能。              |
| **システムプロンプト**     | -              | モデルへの指示内容（Prompt）。このタスクで何をするべきかを自然言語で明記。          |
| **応答表示（ユーザー向け）**  | -              | このモジュールの出力をユーザー画面に表示するかどうかを制御。中間処理用途では非表示推奨。      |
| **応答の創造性（温度）**    | -              | モデル出力のランダム性（創造性）を制御。0に近いと厳密・保守的、1に近いと自由で多様性あり。    |
| **応答文字数上限**       | -              | モデルの出力する最大文字数を制限。モデルによって制限値が異なるため注意。              |
| **応答完了（終了通知）**    | ブール型           | モジュール処理完了時にTrueを出力。フロー制御に使用。                      |
| **応答内容（返信）**      | 文字列型           | モデルの生成した自然言語応答。ユーザーに返すほか、他モジュールへの入力にも利用可能。        |
| **モジュール実行完了**     | ブール型           | モジュール処理完了後に **True** を出力します。通常は次の処理のトリガーとして使用します。 |
